# Lifespan Calorie Burn Dynamics

## Overview

This project analyzes calorie burn patterns across different ages,
activity types, BMI levels, and metabolic characteristics.

## Objectives

- Understand calorie burn behavior
- Compare activity types
- Analyze BMI and RMR impact
- Identify user segments using clustering
- Predict calorie expenditure

## Dataset Features

- Age
- Gender
- Activity Type
- Duration
- Heart Rate
- BMI
- RMR
- Calories Burnt
- Hydration Status
- Sleep Schedule

## Techniques Used

- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis
- Clustering
- Machine Learning

## Key Findings

- Running burns highest calories
- BMI moderately influences calorie expenditure
- RMR strongly impacts calorie burn
- Active users form distinct clusters
